    DarkPapers

    DarkPapers is a tweak that adds over 50 OLED friendly and Full HD day/night wallpapers (with more coming in future updates) into the iOS 13 native dark mode wallpaper cycle (apperance aware).

    No setting up is required, simply install the tweak and go to the wallpaper section in the settings app and look in the still wallpapers to see your new dark mode compatible wallpapers.

    Important Notes

    This package depends on WallpaperLoader in order to work. You can find the tweak in Skittyblock’s repository at:

    https://skitty.xyz/repo

    This tweak is ONLY compatible with iOS 13 and above. The wallpapers included in the package are NOT optimized for iPads, only for iPhones. iPad users can (optionally) download the WallpaperLoader (Prefs) tweak, which is a different tweak from WallpaperLoader, in order to use their own custom wallpapers, or wait until the next update that adds iPad compatibility.

    There's a harmless bug that when you preview one of the custom day and night wallpapers while enabling or disabling dark mode, it will zoom in on the wallpaper, but you can still select it, as it won't be zoomed in once applied, it only appears so in the preview. To fix it, simply stop previwing the wallpaper and then preview it again.
